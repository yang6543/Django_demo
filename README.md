# Django_demo
### 前端
1. **nvm**：用来管理`node.js`的工具。
2. **node.js**：自带有`npm`包管理工具。
3. **npm**：类似于**Python**中的`pip`。可以非常方便的管理一些前端开发的包。
4. **gulp**：用来自动化开发流程。比如`sass`转`css`，`css`和`js`压缩等。
